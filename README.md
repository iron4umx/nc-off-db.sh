# North Carolina Criminal Offender Database Creator

This script is designed to help you create a database of criminal offenders in North Carolina (United States). It automates the process of downloading, organizing, and importing the data into a SQLite database.

## Getting Started

These instructions will help you get started with using the script to create the database.

### Prerequisites

Before running the script, make sure you have the following dependencies installed:

- Bash shell
- wget
- unzip
- gawk
- SQLite 3

### Installation

1. Clone the repository:

2. Make the file nc-off-db.sh executable with: chmod +x nc-off-db.sh

3. Run the script: ./nc-off-db.sh

### Usage

The script will prompt you to specify the working directory for the database and will automatically download, organize, and import the data. Ensure you have at least 50GB of free space in the working directory.
